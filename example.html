📋 集成代码

在您的网站中添加以下代码即可集成 AI 助手：

<!-- 加载 AI 助手 SDK --> <script src="http://localhost:3000/embed.js"></script> <!-- 初始化助手 --> <script> window.initAIAssistant({ config: { position: 'bottom-right', // 位置 theme: 'light', // 主题 enableVoice: true // 启用语音 }, onReady: () => { console.log('AI 助手已就绪！'); }, onError: (error) => { console.error('助手加载失败:', error); } }); </script>

⚙️ 配置选项

// 支持的配置选项 { baseUrl: 'http://localhost:3000', // 服务器地址 position: 'bottom-right', // 位置选项 theme: 'light', // 主题: light/dark/auto enableVoice: true, // 是否启用语音 maxMessages: 50 // 最大消息数 } // 位置选项 - 'bottom-right' 右下角（默认） - 'bottom-left' 左下角 - 'top-right' 右上角 - 'top-left' 左上角
💡 提示： 你现在应该能在页面右下角看到一个橙色的悬浮按钮，点击它就可以开始与 AI 助手对话了！
